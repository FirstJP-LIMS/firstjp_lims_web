Test Codes
  • Use short, unique codes
  • Examples: HGB, WBC, GLU
  • Maximum 64 characters
  • Codes are unique per vendor
Result Types
  • QNT Quantitative - Numerical results
  • QLT Qualitative - Text/Boolean results
Turnaround Time

Format: HH:MM:SS
Example: 4:00:00 for 4 hours

Reference Ranges

Normal Range: Expected values for healthy patients
AMR: Analytical measurement range of instrument
Reportable: Range lab can verify results
Panic: Critical values requiring immediate attention

Field Tips
  • Preparation instructions are required when "Preparation Required" is checked
  • Patient-friendly description should use simple, non-technical language
  • Reference ranges must have min < max
