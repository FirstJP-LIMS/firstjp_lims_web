About Document References
Why Link Documents?
  • Establish relationships between related procedures
  • Track document dependencies
  • Ensure consistency across documentation
  • Support document change impact analysis
Reference Types Explained:
Related General reference for additional information
Supersedes Indicates this document replaces an older one
Supplement Provides additional supporting details
Procedure Points to methods or procedures used
Best Practices:
  • Link documents that are commonly used together
  • Reference parent procedures from child documents
  • Document supersession relationships clearly
  • Include notes when relationship is not obvious
